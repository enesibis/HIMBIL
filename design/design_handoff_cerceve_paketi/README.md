# Hımbıl — Çerçeve Paketi (15 avatar çerçevesi)

Sıcak Karnaval setine ait 15 avatar çerçevesi. Mağaza > Çerçeveler sekmesinde satılır;
takılı çerçeve ana ekran avatarının etrafında görünür.

## İçerik
- `cerceveler-svg/` — 15 çerçeve, vektör SVG (72x72 viewBox)
- `cerceveler-png/` — aynı çerçeveler, 360x360 şeffaf PNG
- `cerceve-sanati.js` — çerçeveleri üreten kaynak kod (parametrik, saf JS + SVG)

## Geometri
- Tuval: 72x72, merkez (36, 36)
- Avatar deliği: yarıçap ~26 (çerçeve bandı r 26–34 arası)
- Uygulamada avatar, kutunun ~%72'si çapında ortalanır; çerçeve SVG/PNG üstüne bindirilir.
